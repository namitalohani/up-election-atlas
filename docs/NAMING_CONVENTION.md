# Naming Convention

## Constituencies

Format:

001-behat.json

174-lucknow-central.json

403-duddhi.json

Rules:

- Use official Uttar Pradesh Assembly Constituency number.
- Lowercase only.
- Words separated with hyphens.
- No spaces.
- No underscores.

## Candidates

firstname-lastname.json

Example:

ravidas-mehrotra.json

## Elections

2022.json

2017.json

2012.json

## Dates

YYYY-MM-DD

Example:

2026-07-03

## IDs

Always use official Election Commission IDs wherever available.
