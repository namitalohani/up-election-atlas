Open Election Atlas of Uttar Pradesh

About the Project

The Open Election Atlas of Uttar Pradesh is an open-source, citation-first database documenting all 403 Assembly constituencies of Uttar Pradesh ahead of the 2027 Assembly Election.

The objective is to build a structured, transparent and publicly accessible repository of electoral information that can be used by students, journalists, researchers, political analysts, policy professionals and citizens.

Rather than publishing isolated reports or social media posts, this project focuses on creating a continuously updated database where every fact is linked to a verifiable source.

⸻

Project Goals

* Build a complete database for all 403 Uttar Pradesh Assembly constituencies.
* Document election history, candidates, demographics, voter statistics and political developments.
* Maintain strict source attribution for every data point.
* Clearly identify missing or conflicting information instead of making assumptions.
* Make the database machine-readable for AI applications.
* Generate educational content, visualisations and election explainers directly from verified data.
* Keep the entire project open source.

⸻

Data Principles

Every entry in this repository follows these rules:

* No fabricated or estimated data.
* Every fact must cite its source.
* Missing information is explicitly marked.
* Conflicting figures are preserved with source references.
* Official government data is preferred wherever available.

Long-Term Vision

The Open Election Atlas aims to become a comprehensive public reference for Uttar Pradesh elections by combining structured electoral data, historical records, demographic information and verified news into a single searchable knowledge base.

The long-term roadmap includes:

* Interactive constituency dashboards
* Election maps
* Candidate profiles
* Historical voting trends
* AI-powered constituency summaries
* Educational explainers
* Public API
* Automated visualisations
* Social media content generated directly from verified data
